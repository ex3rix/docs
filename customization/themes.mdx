Themes control the **visual layout and structure** of your profile. You can choose from three unique designs to match your personal style; to get started, head to [**haunt.gg/dashboard/layout**](https://haunt.gg/dashboard/layout) and choose from:

- `Default`([Example](https://haunt.gg/default))
- `Modern` ([Example](https://haunt.gg/modern))
- `Simplistic`([Example](https://haunt.gg/simplistic))
- `Portfolio`([Example](https://haunt.gg/portfolio))

## Default Theme

The **Default** layout is the most customizable option.

### Features

- Upload a **Profile Banner**
- Choose from **15 social cards**

## Modern Theme

The **Modern** theme offers a sleek, minimalistic design that focuses on social presence.

### Supported Social Integrations

You can select up to **four** of the following:

- Discord Server
- GitHub Profile
- Roblox Profile
- Telegram Channel
- Last.fm Profile
- Stats.fm Profile
- Valorant Profile
- Chess Profile/Club
- TikTok Profile
- Instagram Profile
- X (Twitter) Profile
- Steam Profile
- Spotify Profile
- Discord Bot Profile
- YouTube Profile
- Twitch Profile
- Minecraft Server
- Clash of Clans Profile/Clan
- Brawl Stars Profile/Clan
- Weather

## Simplistic Theme

The **Simplistic** theme provides a clean, content-focused look.

- No additional configuration options
- Great for text-focused profiles or low-distraction layouts

## Portfolio Theme

The **Portfolio** theme is designed for showcasing your work and projects.
It offers various customization options to highlight your skills and projects.

### Features

- Create projects with **title, description, tags, link & icon**
- Choose between different **skills** to showcase what you can do
- Supports [**About Me**](../customization/customization#about-me) feature to show important information about yourself
- Supports [**Social Cards**](../customization/themes#supported-social-integrations), similar to the **Modern theme**
